Back to HollerPro ⚡ 3-minute setup

Make posts appear everywhere, instantly

Right now posts only appear on the device that wrote them. This guide connects HollerPro to a free cloud database so everyone sees everything in real time — no refresh needed.

1
Create a free Firebase project
Firebase is Google's free cloud database — no credit card, no server, nothing to install.

Click the button below to open Firebase Console. Sign in with any Google account, then click "Create a project" (or "Add project").
Opens in a new tab
Open Firebase Console →
💡 On the project creation screen: give it any name (e.g. hollerpro), click Continue, you can turn off Google Analytics if it asks, then click Create project. Wait ~10 seconds for it to finish.
2
Register your web app & copy the config
Inside your new project, look for the </> icon (it says "Web" if you hover it) and click it.

App nickname: type anything, like hollerpro
Click "Register app" — don't tick Hosting.

You'll then see a grey box with code that looks like this:
What you'll see after clicking "Register app"
// Your web app's Firebase configuration
const firebaseConfig = {
  apiKey: "AIzaSy…lots of characters…",
  authDomain: "your-project.firebaseapp.com",
  databaseURL: "https://your-project-rtdb.firebaseio.com",
  projectId: "your-project",
  …
};
⚠️ Important: Select all the text inside the curly braces { … } and copy it. You'll paste it in Step 4. Then click "Continue to console".
3
Turn on the Realtime Database
In the left sidebar of Firebase Console, click Build → Realtime Database.

Click the "Create database" button.

A popup appears asking for a location — pick whichever is closest to you, then click Next.

On the next screen, choose "Start in test mode", then click Enable.
🔓 "Test mode" means the database allows reads and writes without authentication — perfect for a friend group. You can tighten the rules later if you want.
4
Paste your config here

5
Replace firebase-config.js and push to GitHub
Option A — VS Code (easiest):
1. In VS Code, open the file firebase-config.js inside your HollerPro project folder.
2. Select everything (Ctrl+A), then paste what you copied (Ctrl+V).
3. Save the file (Ctrl+S).
4. Open the Source Control panel (left sidebar icon that looks like a branch), type a commit message like "add firebase config", click Commit, then Sync Changes.

Option B — GitHub website:
1. Go to github.com/PepeGoat/HollerPro
2. Click on firebase-config.js → click the pencil ✏️ edit icon (top right of the file view).
3. Select everything and paste your copied content.
4. Scroll down → click "Commit changes".
🕐 GitHub Pages takes 1–2 minutes to rebuild after you push. Then visit pepegoat.github.io/HollerPro and posts will appear on every device instantly.
